git init - создаем git репозиторий из текущей папки 
git add . - просто пишем эту комманду каждый раз перед  commit 
git commit -m 'message' - создаем новую версию 
 git push origin master - закидываем версию в github
git pull origin master- стягиваем ноыве данные с гитхаб (если они есть в гитхаб , но их нет в локальном)
git clone ssh -ссылка -  копируем репозиторий с гита
git log - посмотреть все коммиты

git branch - показывает все ветки (* обозначено ветка в которой мы находимся сейчас)
git branch название - создать новую ветку(просто создаем, но не переходим)
git branch -D  ветка - удалить ветку
git checkout ветка - перейти на ветку
git checkout -b название - создать ветку и перейти на неё

git reset HEAD~
git restore .
